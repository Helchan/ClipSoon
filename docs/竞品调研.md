# ClipSoon 竞品调研与取舍

调研日期：2026-07-19。仅参考产品官网、官方文档和官方仓库。

## 结论

ClipSoon 采用“Raycast 的呼出与键盘流、Paste 的视觉预览、Maccy 的轻量、Ditto 的直接粘贴、
PastePal 的图片/文件辨识、CopyQ 的跨平台与隐私边界”。首版保持本地、无账号、无云和无遥测，
不因追求竞品功能数量而牺牲响应速度。

## 系统基线

- [Apple Spotlight 剪贴板历史](https://support.apple.com/en-ie/guide/mac-help/mchl40d5b86b/mac)
  验证了居中搜索浮层和敏感内容提醒；ClipSoon 少一步，选中后直接回原应用粘贴。
- [Windows Win+V](https://support.microsoft.com/en-us/windows/apps/using-the-clipboard) 是原生基线，
  但历史数量、文件支持和检索能力有限，ClipSoon 提供可搜索的持久本地历史。

## 形成的产品契约

- 当前屏幕上部居中；搜索立即聚焦；再次热键、Esc、外部点击、失焦和发送后隐藏。
- 列表只显示高辨识度摘要，右侧显示大图、完整文本或多文件路径。
- 文件、图片、文本采集优先级固定为文件 > 图片 > 文本。
- 发送失败时必须区分“已写入剪贴板”和“已确认粘贴”，不误报。
- 搜索采用项目自定义的确定性顺序匹配公式，以满足可解释、可测试和极低延迟；没有照搬竞品
  不透明的 frecency 排序。
- 本地 SQLite + PNG、无网络；尊重平台 transient/concealed/禁止历史标记，提供暂停和清除。
