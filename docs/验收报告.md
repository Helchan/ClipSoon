# ClipSoon 0.9.0 验收报告

验收日期：2026-07-19
本机环境：macOS 15.7.7、Apple Silicon arm64、Python 3.12.10

## 1. 结论

核心功能、搜索契约、持久化、隐私格式、快捷键状态机、Qt 界面和 macOS Apple Silicon 应用包
均已完成本机验证。Windows 代码路径和构建脚本已提供并有自动化边界测试，但没有 Windows
主机，因此不能将本次结果表述为 Windows 真机通过；Intel macOS 也未生成 universal2 产物。

## 4. macOS 产物

- 产物：`dist/ClipSoon.app`
- 架构：arm64
- 版本：0.9.0（build 1）
- 窗口模式：`LSUIElement=true`，不占用 Dock
- 签名：ad-hoc；`codesign --verify --deep --strict` 通过
- 启动策略：PyInstaller one-dir，避免 one-file 每次解包

该包尚未使用 Apple Developer ID 签名和公证，不应作为面向陌生用户的正式发行包。

## 5. UI 与交互验收

真实 macOS 界面已检查 Spotlight 式居中浮层、搜索自动聚焦、文本/截图/文件筛选、图片缩略图
和大图预览、单个文本文件的限量只读预览、完整文件路径、深浅主题设置、`Esc` 隐藏、精确匹配
置顶及乱序查询排除。文件图标在真实滚动绘制中发现并修复了 `QFileIconProvider` 参数类型问题，
修复后文件预览正常；README.md 实际验证只展示前 220 个字符，`...` 在预览框内可见，向下滚动
后内容位置不变，且不显示滚动条或打开编辑器。

最终源码另外通过自动化验证设置页显示物理 Control 语义、Windows Meta round-trip，以及
“取消 / 保存”本地化。macOS 辅助功能权限未由验收过程擅自开启，因此全局双 Ctrl 和跨应用
自动粘贴只验证了状态机、目标恢复与降级逻辑，未宣称完成有权限条件下的端到端验收。
